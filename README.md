# Serdar Hoca GYM Fitness Club Web Sitesi

Bu proje bağımlılık gerektirmeyen statik bir web sitesidir. `index.html` dosyasını tarayıcıda açarak çalıştırabilirsiniz.

## Medya Dosyaları

Site şu anda `assets/images` klasöründeki yüklenmiş salon fotoğraflarını ve `assets/videos/IMG_5259.MOV` videosunu kullanır.

MOV dosyası 4K ve yaklaşık 1 GB olduğu için ana ekranda otomatik arka plan videosu olarak yüklenmez. Video, tanıtım bölümünde özel oynatıcı içinde yer alır; orijinal sesi kapalıdır ve site tarafında telifsiz/orijinal gym ritmi üretilir. Oynatma 1 dakika 32 saniyede durur.

Yayına çıkmadan önce bu videonun 720p/1080p H.264 `.mp4` sürümüne çevrilmesi önerilir. Mevcut çalışma klasöründe shell üzerinden yeni medya dosyası yazma izni olmadığı için dönüştürülmüş MP4 dosyası proje içine otomatik kaydedilemedi.
